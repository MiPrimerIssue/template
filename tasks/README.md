# Tareas del proyecto

En esta carpeta se organizan las tareas disponibles para contribuir al proyecto.

## Niveles

- `beginner/` → tareas sencillas para empezar
- `intermediate/` → tareas con algo más de complejidad
- `advanced/` → tareas más técnicas o de mayor alcance

## Estructura de una tarea

Cada tarea debe incluir:

- título
- nivel
- objetivo
- contexto
- requisitos
- archivos sugeridos
- criterios de aceptación

## Importante

No empieces una tarea sin haberla solicitado y haber sido asignado desde la plataforma MiPrimerIssue.
