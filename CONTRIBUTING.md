# Guía de contribución

Gracias por tu interés en contribuir a este proyecto.

Este repositorio está diseñado para que desarrolladores junior puedan participar en proyectos open source reales de manera guiada.

---

## Antes de empezar

Antes de contribuir:

- Lee el `README.md`
- Revisa la carpeta `tasks/`
- Solicita la tarea desde la plataforma MiPrimerIssue
- Espera a ser asignado antes de comenzar

---

## Flujo de contribución

1. Entra en MiPrimerIssue
2. Solicita una tarea disponible
3. Cuando se te asigne, revisa las instrucciones
4. Haz fork del repositorio
5. Crea una rama para tu tarea
6. Realiza los cambios
7. Sube tu rama a tu fork
8. Abre una Pull Request al repositorio principal

---

## Convención de nombres para ramas

Usa un nombre de rama descriptivo. Ejemplos:

```bash
feat/task-001-search-bar
fix/task-010-login-bug
docs/task-020-readme-update
