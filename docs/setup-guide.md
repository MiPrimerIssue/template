# Guía de configuración

## Requisitos previos

Antes de empezar asegúrate de tener instalado:

- Node.js
- npm o pnpm
- Git

---

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/TU_ORG/NOMBRE_DEL_REPO.git
cd NOMBRE_DEL_REPO

Instala dependencias:

npm install

Copia las variables de entorno:

cp .env.example .env.local

Rellena las variables necesarias y ejecuta:

npm run dev
