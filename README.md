# 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/TU_ORG/NOMBRE_PROYECTO.git
cd NOMBRE_PROYECTO

Instala dependencias:

npm install

Ejecuta el proyecto:

npm run dev

Abrir en el navegador:

http://localhost:3000


📁 Estructura del proyecto

src/
  app/ → páginas y rutas
  components/ → componentes reutilizables
  lib/ → utilidades
  types/ → tipos de TypeScript

tasks/
  beginner/ → tareas para nuevos contribuidores

docs/
  documentación del proyecto
